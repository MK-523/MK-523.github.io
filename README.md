# Mahesh Karthikeyan — Personal Website

Personal portfolio for [Mahesh Karthikeyan](https://github.com/MK-523), deployed through GitHub Pages.

The root contains the production build served by GitHub Pages. The editable React/Vite source lives in `src/`; rebuild with `npm run build` and publish the contents of `dist/` to the repository root.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
